For compatibility, Como suports two modes to apply your code: Full & Only

Full mode: It extends JS's own object, eg. String, Function, Array, Date, Object, and Class, $. Then you can write code like " ' abc '.trim(); $('#test'); "

Only mode: It doesn't extend JS's own object. All function of object is belong to Como. So you must write code like " Como.String.trim(' abc '); Como("#test"); "
When you use these object's functions, the first parameter is the object. 

Full: function(){}.bind(this, p1, p2);
Only: Como.Function.bind(function(){}, this, p1, p2);

The default mode is Full. If you have used third framework such as JQuery, you should user the Only mode.

How define mode? Plus '?only' to the end of Como file's path.

<script type="text/javascript" src="/js/Como.js?only"></script>
