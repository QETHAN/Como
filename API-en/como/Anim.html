Anim
Animation Object.
to by from duration checkpoint blind hide show ondone go stop ease
//define an animation object
new Como.Anim($("#test"));
Como.anim($("#test"));
$("#test").anim()

//the anim object can change most of the css attribute. 

//see following.
to('backgroundColor', '#000000');
to('borderColor', '#000000');
to('color', '#000000');
to('opacity', 0.5);

to('width', 100), ('width', '100px');
to('scrollLeft', '100px');
to('scrollTop', '100px');
to('margin', '10px 5px');
to('padding', '10px 5px');
borderWidth,lineHeight,fontSize,marginBottom,marginLeft,marginRight,marginTop,paddingBottom,

to
change the attribute to the value. to(css, value)
Como.anim($("#test"))to('height', '50px').go();
$("#test").anim().to('height', '50px').go();		//current height is 50px
by
change the attribute by the value. by(css, value)
$("#test").anim().by('height', '50px').go();		//current height = old height + 50px
from
change the attribute from the value
$("#test").anim().from('height', '10px').to('height', '50px').go();
duration
set the time when the animation end. 500 millisecond is the default value. (millisecond)
$("#test").anim().to('height', '50px').duration(1000).go();
checkpoint
when finish some animation, checkpoint, and then run the next.
$("#test").anim().to('height', '100px').checkpoint().to('height', '50px').go();
hide
hide the element
$("#test").anim().to('opacity', 0).blind().hide().go();
show
show the element
$("#test").anim().to('opacity', 1).blind().show().go();
ondone
when the animation finished, the ondone will be called. you can give ondone a function (the first argument is the element(como)). 
$("#test").anim().to('height', '50px').ondone($("#test").width(50)).go();
go
run the animation.
$("#test").anim().to('height', '50px').go();
stop
stop the animation. if there is an paramater and it is true, then the animation is finished immediately.
$("#test").anim().to('height', '50px').go().stop();
ease
set the animation type of playing, 3 values: begin(from slow to fast), both(from slow to fast, then slow down), end(from fast to slow);
like this: $('#test').anim().to('height', 50).ease(Como.Anim.ease.begin).go(); 
$("#test").anim().to('height', '50px').ease(Como.Anim.ease.begin).go().stop();
