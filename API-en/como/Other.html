Como.template
$.template can realize String template by '{}'.
Running $.template(string), and return an object. the object has two method: set() and run().
You can set value by the set method, just like set('name', 'kevin') or set({'name': 'kevin'}).
After setted all value, run the run method, then you can take the last string after template.
There is the demo.
$.template('my name is {name}, and i am {age}').set({'name': 'kevin', 'age': 25}).run();
If you have the special chars, { and }, in the templated string, you can use '\\' to change it. 
var test = Como.template("my name is {name}, i am \\{sex\\}, i am {old} years old!");
test.set('name', 'kevin').set('old', 25).set('sex', 'body');
//also can be: test.set({'name':'kevin', 'old': 25});
log(test.run());
